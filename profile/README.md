## HytaleModding

We are a community that wants to support new and experienced mod developers to mod Hytale. We provide free resources, guides and documentation for Hytale. 

### Code Contributions

Thank you for your interest in contributing to HytaleModding, here are some repositories we are currently accepting contributions to:

- HytaleWebsite: https://github.com/HytaleModding/site
- Discord Bot: https://github.com/HytaleModding/robot
- HyMods Website (external): https://github.com/HytaleModding/HyMods
